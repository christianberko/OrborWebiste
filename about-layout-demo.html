About Us Section Layout Options

How to Use These Layouts:

To switch between layouts, simply change the class on the About Us section in your index.html file:

  • Current: class="about-section split-layout"
  • Option 1: class="about-section centered-card"
  • Option 2: class="about-section hero-background"
  • Option 3: class="about-section timeline-layout"
  • Option 4: class="about-section magazine-layout"

Current: Split Layout

A traditional side-by-side layout with image on the left and text on the right. Features a glass-morphism effect on the text container with subtle shadows and backdrop blur.

  • Clean, professional appearance
  • Good balance between image and text
  • Responsive design that stacks on mobile
  • Subtle background patterns and gradients

Option 1: Centered Card Layout

A modern, centered card design with the image as a circular profile picture and text centered below. Perfect for a more personal, approachable feel.

  • Circular profile image with blue border
  • Centered text layout for better readability
  • Glass-morphism card container
  • Clean, modern aesthetic

Option 2: Hero Background Layout

A dramatic hero-style layout with the staff image as a full background and text overlay in a glass card. Creates a more immersive, storytelling experience.

  • Full-screen background image
  • Parallax scrolling effect
  • Glass-morphism text overlay
  • Dramatic, immersive feel

Option 3: Timeline Layout

A creative timeline-style layout with a central line connecting the image and text sections. Features timeline markers and a unique visual flow.

  • Central timeline line with gradient
  • Timeline markers on text section
  • Positioned elements for visual flow
  • Creative, unique design approach

Option 4: Magazine Layout

A sophisticated magazine-style layout with larger typography, elegant spacing, and a subtle geometric background pattern. Perfect for a premium, editorial feel.

  • Large, bold typography
  • Subtle geometric background patterns
  • Image with offset shadow effect
  • Premium, editorial aesthetic
